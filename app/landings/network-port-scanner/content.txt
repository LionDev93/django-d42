Comprehensive Network Port Scanner


[landing.content]
The Device42 network port scanner enables SNMP, WMI, and SSH discovery, comprehensively identifying and tracking all of your physical and logical components, relationships, and interdependencies!

- Network port scanner features complete hardware & software scan - Windows, Linux, *NIX
- Full network port scanning: Switches, Routers, Firewalls, Load Balancers
- IP to MAC address and MAC to switch port relationships are discovered by Device42's network port scanner
- Network port scanner discovers VLANs, subnets, and CDP / LLDP neighbors
- Enjoy automatic cloud & hypervisor discovery with custom network port scanner schedules!
