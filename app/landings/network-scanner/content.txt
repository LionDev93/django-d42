[landing.title]
Comprehensive Network Scanner Software

[landing.image_format]
png

[landing.content]

Device42's network scanner tools automatically discover and generate a centralized repository of application, service, and device relationships and dependencies!

- The Device42 network scanner is agent-less, and discovers your key dependencies using native protocols: WMI for Windows, SSH for Linux, and SNMP.
- Device42's network scanner discovers all of your deployed software and hardware details, including applications, dependencies, manufacturers, and versions!
- All services dependencies are automatically discovered, running or not, including their protocols & ports.
- Easily view application to server and application to application dependencies.
- Device42 network scanner builds detailed application dependency maps even for machines that can't be directly accessed using Netflow data!
