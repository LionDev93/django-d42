Comprehensive Agentless Cloud Application Discovery Tool


[landing.content]
The Device42 cloud application discovery tool enables native hypervisor, SNMP, WMI, and SSH discovery, comprehensively identifying and tracking all of your physical and logical components, relationships, and interdependencies!

- Complete host and guest discovery: hardware & software on Windows, Linux, and *NIX/BSD
- Full network auto-discovery: switches, routers, firewalls, load balancers
- Support for all common hypervisors: VMware, Hyper-V, Xen, KVM
- Discover VLANs, subnets, and CDP / LLDP neighbors
- Complete cloud and hypervisor discovery with custom Cloud Application Discovery schedules - Scan your entire IT infrastructure in minutes!
